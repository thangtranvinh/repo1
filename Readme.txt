1. Sign in to the Azure portal (https://portal.azure.com).

2. In the Azure portal, search for "Key Vaults" and select "Key Vaults" from the results.

3. Click on the "Add" button to create a new Key Vault.

4. Provide a unique name for the Key Vault, select the subscription, resource group, and region.

5. Enable the "Soft delete" option if you want to enable recovery of deleted keys.

6. Click on the "Review + Create" button and then click on "Create" to create the Key Vault.

7. Once the Key Vault is created, navigate to the Key Vault resource.

8. In the Key Vault, click on the "Keys" option in the left-hand menu.

9. Click on the "Generate/Import" button to create a new key.

10. Provide a unique name for the key, select the key type (RSA or EC), and specify the key size or curve.

11. Choose the key operations and key usage permissions as per your requirements.

12. Click on the "Create" button to create the key.

13. Once the key is created, you can access it using the Key Vault's URL and authenticate with appropriate credentials.

14. Make sure to securely manage and protect the access to the Key Vault and its keys.

15. You can now use the created key in your applications or services that require encryption or decryption operations.
